# Framework

Storybook is architected to support diverse web frameworks, including React, Vue, Angular, Web Components, Svelte, and over a dozen others. This section helps you get started on adding new framework support for Storybook Rsbuild.

:::info
Check out more details on [Contributing a Storybook framework](https://storybook.js.org/docs/contribute/framework).
:::
