# Web Components

Storybook for Web Components & Rsbuild is tailored for Lit and custom-element projects, giving you fast component previews without leaving your Rsbuild workflow.

## Requirements

<div style={{ width: '100%', maxWidth: '400px' }}>

| Package | Version |
| :--- | :--- |
| `lit` | ≥ 2 (or ≥ 3) |
| `@rsbuild/core` | ≥ 1.0.1 |
| `storybook` | ≥ 10.0.0 |

</div>

## Getting started

Run through these steps inside an Rsbuild project that renders Web Components.

### Installation

Install the Web Components framework package.

<PackageManagerTabs command="install storybook-web-components-rsbuild -D" />

### Configure `.storybook/main.ts`

```js
import { StorybookConfig } from 'storybook-web-components-rsbuild'

const config: StorybookConfig = {
  framework: 'storybook-web-components-rsbuild',
  rsbuildFinal: (config) => {
    // Customize the final Rsbuild config here
    return config
  },
}

export default config
```

You're good to go! See the [web components sandbox](https://github.com/rspack-contrib/storybook-rsbuild/blob/main/sandboxes/web-components) for a complete reference and browse the [Storybook docs](https://storybook.js.org/docs) for deeper guidance.
