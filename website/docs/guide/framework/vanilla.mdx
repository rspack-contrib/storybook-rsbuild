# Vanilla JS / TS

Use Storybook for vanilla JavaScript or TypeScript projects to preview UI components while keeping your Rsbuild pipeline intact.

## Requirements

<div style={{ width: '100%', maxWidth: '400px' }}>

| Package | Version |
| :--- | :--- |
| `@rsbuild/core` | ≥ 1.0.1 |
| `storybook` | ≥ 10.0.0 |

</div>

## Getting started

These steps assume you already have an Rsbuild project without a specific framework.

### Installation

Install the HTML framework package to enable vanilla stories.

<PackageManagerTabs command="install storybook-html-rsbuild -D" />

### Configure `.storybook/main.ts`

```js
import { StorybookConfig } from 'storybook-html-rsbuild'

const config: StorybookConfig = {
  framework: 'storybook-html-rsbuild',
  rsbuildFinal: (config) => {
    // Customize the final Rsbuild config here
    return config
  },
}

export default config
```

You're all set! Review the [vanilla TypeScript sandbox](https://github.com/rspack-contrib/storybook-rsbuild/blob/main/sandboxes/vanilla-ts) to see the setup in action, and continue exploring the [Storybook docs](https://storybook.js.org/docs).
